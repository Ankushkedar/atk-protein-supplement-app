║                            QUICK START                                       ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. Install Node.js from https://nodejs.org/

2. Open terminal in project folder and run:
   npm run install-all

3. Setup environment:
   cd server
   copy .env.example .env
   (Edit .env with your MongoDB connection)

4. Run the application:
   cd ..
   npm run dev

5. Open browser:
   http://localhost:3000
