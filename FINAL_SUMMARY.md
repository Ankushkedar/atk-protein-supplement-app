## 🔧 Customization Guide

### Change Brand Name
**File:** `client/src/components/Navbar.js`
**Line:** 18
```javascript
<span className="logo-text">YOUR BRAND NAME</span>
```

### Change Colors
**File:** `client/src/index.css`
**Lines:** 23-27
```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
}
```

### Add Your Logo
1. Save logo as `client/public/logo.png`
2. Update Navbar.js to use image instead of text

### Add More Products
**File:** `client/src/pages/Products.js`
**Lines:** 12-100
Add to `sampleProducts` array

### Update Contact Info
**Files:**
- `client/src/components/Footer.js`
- `client/src/pages/Contact.js`
